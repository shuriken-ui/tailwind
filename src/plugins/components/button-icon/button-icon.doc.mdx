# Button Icon

Buttons are an essential part of any application. Shuriken UI icon buttons can have different shapes and sizes. They can be used as links or as HTML buttons.

## State

### State: loading

Icon buttons can be shown in a loading state. The loading indicator informs the user that something is being loaded.

<div className="flex items-end gap-2 bg-slate-100 dark:bg-slate-900 p-6 rounded-sm">
  <Story of={ButtonIconStories.SizeSmLoading} />
  <Story of={ButtonIconStories.SizeMdLoading} />
  <Story of={ButtonIconStories.SizeLgLoading} />
</div>

<br />

### State: disabled

Icon buttons can be shown in a disabled state. They are a bit opaque and cannot be clicked to peform an action.
