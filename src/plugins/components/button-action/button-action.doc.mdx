# Button Action

Buttons are an essential part of any application. Shuriken UI action buttons can have different shapes and colors. They can be used as links or as HTML buttons.

## State

### State: loading

Action buttons can be shown in a loading state. The loading indicator informs the user that something is being loaded.

### State: disabled

Action buttons can be shown in a disabled state. They are a bit opaque and cannot be clicked to peform an action.
